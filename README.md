# 🚀 AI Document Intelligence System

An advanced AI-powered application that processes, analyzes, and summarizes documents across multiple formats.  
It supports multilingual processing, intelligent querying, and even generates video summaries from documents.

---

## 📌 Features

- 📄 Multi-format document processing (PDF, Word, Excel, PPT, Images, Audio, Video, etc.)
- 🌐 Multilingual support (English, Hindi, Kannada)
- 🤖 AI-powered summarization and Q&A
- 🔍 Semantic search using RAG (Retrieval-Augmented Generation)
- 🎥 Automatic video generation from document summaries
- 🔊 Text-to-Speech (TTS) with multilingual narration
- 📊 Chat analytics and performance tracking

---

## 🧠 How It Works

1. Upload a document  
2. System extracts and processes content  
3. AI generates summaries and answers queries  
4. Optional: Convert summary into an animated video  

---

## 🛠️ Tech Stack

- **Frontend:** React / TypeScript  
- **Backend:** Python  
- **AI/ML:** OpenAI / Local LLM (Ollama)  
- **Database:** SQLite + ChromaDB  
- **Other Tools:** LangChain, Whisper, gTTS
